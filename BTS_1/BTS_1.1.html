Domande esercizio 1.1

1) Bootstrap è un framework per la creazione del front-end di una pagina web. Contiene anche elementi HTML già pronti con aggiunte di JS.

2) Per usare i servizi di Bootstrap bisogna includere nell'header del codice la seguente riga: <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">, mentre alla fine del body bisogna aggiungere: <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>.

3) Una CDN (Content Delivery Network) sono un insieme di server situati in varie zone del mondo che forniscono contenuti per i siti web velocemente. Molti utenti in verità hanno già bootstrap scaricato perche` esso viene scaricato quando si apre un sito. Quindi verra´ caricato dalla cache ottimizando i tempi.

4) "width=device-width" adatta la grandezza della pagina in base alle dimensioni dello schermo. "initial-scale=1.0" Indica il livello dello zoom quando viene caricata per la prima volta la pagina.

5) Il container è un contenitore responsivo con una larghezza predefinta. Il container-fluid invece ha una larghezza uguale alla larghezza della schermo.
